# プロジェクト指示書: zip2Addr

## プロジェクト概要
`zip2Addr` は、シンプルでモダンなウェブベースの郵便番号検索ツールです。ユーザーは7桁の郵便番号（ハイフンあり・なし両対応）を入力することで、対応する住所と読みがな（カナ）を取得・表示できます。

- **アーキテクチャ:** クライアントサイド・ウェブアプリケーション（シングルページ）
- **主要技術:**
  - **フロントエンド:** HTML5, Vanilla JavaScript (ES6+)
  - **スタイリング:** Tailwind CSS (CDN経由)
  - **外部API:** [ZIPCloud](https://zipcloud.ibsnet.co.jp/) (住所検索用)

## ディレクトリ構成
- `src/`: コアアプリケーションのロジックとアセット。
  - `index.html`: メインのエントリポイントとUI構造。
  - `script.js`: API連携、正規化、DOM操作のコアロジック。
  - `style.css`: カスタムスタイルのプレースホルダー（主にTailwindを使用）。
- `01要件定義.md` ～ `04検証手順.md`: 詳細なプロジェクトドキュメント（日本語）。

## ビルドと実行
静的なウェブアプリケーションであるため、ビルドステップはありません。

- **実行方法:** 任意のモダンなウェブブラウザで `src/index.html` を直接開きます。
- **テスト:** UIに様々な郵便番号を入力して手動で検証します。
  - 形式: `1000001` または `100-0001`。
- **リンター/フォーマッタ:** 現在、自動ツールは設定されていません。既存のVanilla JSのパターンに従ってください。

## 開発規約
- **正規化:** API呼び出しの前に、必ず郵便番号からハイフンを除去してください。
- **バリデーション:** 正規化後、入力が正確に7桁の数字であることを確認してください。
- **UI/UX:**
  - 読みがな（カナ）は、漢字住所の上に表示してください。
  - エラーメッセージには赤色のテキストを使用してください。
  - 検索の実行は、「検索」ボタンと「Enter」キーの両方に対応させてください。
- **APIハンドリング:** 非同期データの取得には `fetch` APIを使用してください。エラーハンドリングのために適切な `try/catch` ブロックを確保してください。

## 主要ファイル
- `src/script.js`: 検索リクエストのライフサイクル全体を処理する `searchAddress` 非同期関数が含まれています。
- `src/index.html`: Tailwind CSSクラスを使用してレイアウトを定義しています。
