# 要件定義
以下のポイントを踏まえ、要件定義書を作成して下さい
 - 郵便番号を入力して、検索ボタンを押すと、下部に郵便番号に紐づく住所が表示されるwebアプリを作成する
 - 顧客はエンジニアではないため、非エンジニアでもわかるように要件定義書を作成し、01要件定義.mdに出力して下さい

 # 設計仕様
 01要件定義.mdの内容を踏まえて、設計仕様を作成し、02設計仕様.mdに出力して下さい。
 - 入力した郵便番号に紐づく住所の検索はWeb APIを利用
 - Web APIの使用方法も記載
 - html + css + jsで実現する

  # 実装手順
  02設計仕様.mdの内容を踏まえて、実装手順を作成し、03実装手順.mdに出力して下さい。
  - Step By Stepで手順を記載
  - コードの記載は不要
  - コードの作成や修正はこの段階で不要

  # 実装
  03実装手順.mdの内容を踏まえて、実装してください。
  - ソースコードのフォルダはsrcを使用
  - 必要に応じて、src配下にさらにフォルダを分けてもかまいません
  - cssの作成はできる限りモダンなデザインにしてください。（例：Shadcn UI + Tailwind CSSを使用したりとか）

  # 検証手順
  03実装手順.mdの内容を踏まえて、検証するためのテストケースを作成し、検証までやってください。
  - 正常系、異常系のパターンを分ける
  - 検証結果を記入
