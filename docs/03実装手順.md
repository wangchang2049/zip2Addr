# zip2Addr 実装手順書

本アプリケーションを開発するためのステップバイステップの手順です。

## ステップ 1: プロジェクトの準備
1. 作業用のフォルダを作成します。
2. 必要なファイル（HTML、CSS、JavaScript）を新規作成します。

## ステップ 2: HTMLの構築 (構造の作成)
1. 画面の基本的な骨組み（HTMLファイル）を作成します。
2. 郵便番号を入力するための入力欄、検索ボタン、住所を表示するエリアを配置します。

## ステップ 3: CSSの作成 (デザインの適用)
1. スタイルシート（CSSファイル）を作成し、HTMLファイルに読み込ませます。
2. 郵便番号入力欄やボタンのサイズ、レイアウトを調整し、使いやすい画面にします。

## ステップ 4: JavaScriptの実装 (検索機能の開発)
1. スクリプトファイル（JavaScriptファイル）を作成し、HTMLファイルに読み込ませます。
2. 検索ボタンを押したとき、または郵便番号入力欄でEnterキーを押したときに実行される検索処理関数を定義します。
3. 郵便番号のハイフンを除去する正規化処理と、7桁の半角数字かどうかのバリデーション処理を実装します。
4. JavaScriptから郵便番号検索APIへ問い合わせ（通信）を行う処理を作成します。
5. APIから返ってきたデータから、漢字住所とカナ表記を抽出し、住所表示エリアに反映する処理を作成します。

## ステップ 5: エラー対応と確認
1. バリデーションエラー、APIエラー、住所が見つからない場合の処理を作成します。
2. エラー時は住所表示エリア内に大きな赤文字でメッセージを表示します。
3. 実際にブラウザで画面を開き、ハイフンあり/なしの入力やEnterキー操作を含め、動作確認を行います。
