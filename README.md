# 郵便番号検索Webアプリケーション

このアプリケーションは、郵便番号を入力することで、対応する住所（漢字およびカナ）を素早く確認できるWebツールです。

## 特徴
- **柔軟な入力**: 7桁の郵便番号（`1000001`）と、ハイフン付き（`100-0001`）の両方に対応しています。
- **直感的な操作**: 検索ボタンのクリックはもちろん、入力欄でEnterキーを押すことでも検索が可能です。
- **わかりやすい表示**: カタカナの住所を漢字住所の上に表示し、視認性を高めています。
- **モダンなデザイン**: Tailwind CSSを使用し、シンプルかつ現代的なUIを実現しています。

## 使い方
1. 本リポジトリの `src/index.html` をWebブラウザで開きます。
2. 郵便番号を入力欄に入力します。
3. 「検索」ボタンを押すか、キーボードの「Enter」キーを押します。
4. 画面下部に住所と住所（カナ）が表示されます。

## 技術スタック
- **構造**: HTML5
- **スタイリング**: Tailwind CSS (CDN利用)
- **ロジック**: Vanilla JavaScript (ES6+)
- **郵便番号検索API**: [ZIPCloud](https://zipcloud.ibsnet.co.jp/)

## ディレクトリ構成
- `src/`: ソースコード一式
  - `index.html`: 画面の骨組み
  - `script.js`: 検索・API通信ロジック
  - `style.css`: カスタムCSS（必要な場合）
- `01～04_*.md`: 各工程のドキュメント類
